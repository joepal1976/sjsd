SJSD.Config.Options

The options object is the end result after having parsed the configuration and the command line arguments. It is normally available under [config object].options.

Properties

config
The path to the config file (if any)
debug
Whether to print debug output during the process
files
An array of filenames to parse
outputdir
The path to a directory where to write the output
spam
Whether to spam an excessive amount of debug output during the process
templatesdir
The path to the dir where templates are found
